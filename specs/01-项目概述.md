# 项目概述

**TokenginX** 是一个专为单点登录（SSO）优化的高性能会话存储系统，使用 Go 语言开发。它在 SSO 架构中扮演与 Redis 类似的角色，但针对会话管理进行了深度优化，内置支持多种 SSO 协议。

**开源项目**：本项目遵循 GitHub 标准的开源开发流程，包括代码提交规范、CI/CD 流程和社区协作。

## 核心定位

- **轻量级**：单文件执行，零外部依赖
- **高性能**：100K+ QPS，P99 < 1ms，P999 < 5ms
- **多协议**：原生支持 OAuth 2.0/OIDC、SAML 2.0、CAS
- **多接口**：TCP (RESP)、gRPC、HTTP/REST 三种通信方式
- **智能存储**：内存优先 + 冷数据自动溢出到磁盘

## 与 Redis 的关键区别

- 专为 SSO 会话场景优化，内置协议支持
- 原生支持国密算法（SM2、SM3、SM4）
- 智能冷热数据分离，无需手动配置
- 单二进制部署，无需额外安装配置

## 命令行包

### tokenginx-server

服务器端应用，负责会话存储和管理。支持 Prometheus 监控和 syslog 日志。

### tokenginx-client

客户端工具，用于连接服务器进行监控和查询统计。
