# 性能与版本

## 性能目标

- **QPS**：单节点 > 100,000 次/秒
- **延迟**：
  - P99 < 1ms
  - P999 < 5ms
- **缓存命中率**：> 99%
- **内存效率**：零拷贝优化，最小化内存分配

## 版本规划

- **v0.1.0** (MVP)：单机 + OAuth 2.0 基础支持
- **v0.5.0**：完整协议支持（SAML + CAS + 三接口）
- **v1.0.0**：生产可用（国密 + 安全加固）
- **v2.0.0**：分布式集群 + 高可用
- **v3.0.0**：企业版（Prometheus 监控 + 运维工具）
